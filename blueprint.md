
# Blueprint: 고급스러운 가계부 애플리케이션

## 1. 프로젝트 개요

이 프로젝트는 HTML, CSS, JavaScript를 사용하여 사용자가 수입과 지출을 손쉽게 추적하고 관리할 수 있는 모던한 웹 기반 가계부 애플리케이션을 구축하는 것을 목표로 합니다. 모든 데이터는 브라우저의 `localStorage`에 저장되어 영속성을 가집니다.

## 2. 디자인 및 기능 명세

### 핵심 기능
- **수입/지출 기록**: 날짜, 내용, 금액을 포함한 거래 내역 추가
- **월별/연간 요약**: 월별 및 연간 총수입, 총지출, 순익을 시각적으로 제공
- **상세 내역 확인**: 월별 요약을 클릭하여 일별 상세 내역 확인
- **데이터 관리**: 개별 또는 전체 거래 내역 삭제
- **데이터 영속성**: `localStorage`를 사용한 데이터 보존

### 디자인 시스템 (라이트 모드)
- **색상 팔레트**:
  - 배경: 부드러운 질감의 밝은 회색 (`#F4F7FC`)
  - 컴포넌트 배경: 깨끗한 흰색 (`#FFFFFF`)
  - 강조 색상: 생동감 있는 파란색 (`#007BFF`) 및 녹색 (`#28A745`)
  - 텍스트: 가독성을 위한 어두운 회색 (`#333333`) 및 검정색
- **타이포그래피**:
  - Google Fonts의 'Poppins'를 사용하여 현대적이고 깔끔한 느낌을 제공
  - 명확한 정보 계층을 위한 다양한 폰트 크기 및 굵기 사용
- **레이아웃**:
  - Flexbox를 사용한 반응형 및 유연한 레이아웃
  - 컴포넌트 간 충분한 여백으로 시각적 편안함 제공
- **시각 효과**:
  - 카드형 UI에 부드러운 `box-shadow`를 적용하여 입체감 부여
  - 인터랙티브 요소(버튼, 입력창)에 "glow" 효과 및 부드러운 전환 애니메이션 추가
- **아이콘**:
  - Font Awesome을 활용하여 사용자 인터페이스의 직관성 강화 (예: 추가, 삭제, 수입/지출 아이콘)
- **질감**:
  - 배경에 미세한 노이즈 텍스처를 추가하여 고급스러운 촉감 부여

## 4. 이번 변경 요청: "화이트 모드로 변경해줘" 실행 계획

1.  **CSS 색상 변수 수정**:
    -   `style.css` 파일의 `:root` 선택자에서 다크 모드 색상 변수들을 라이트 모드에 맞는 새로운 색상 값으로 변경합니다.
    -   배경, 텍스트, 컴포넌트, 그림자 등 모든 색상 관련 변수를 업데이트하여 전체적인 테마를 전환합니다.
2.  **HTML 및 JavaScript 코드 확인**:
    -   `index.html` 내의 JavaScript 코드에서 동적으로 색상을 변경하는 부분이 있다면, CSS 변수를 사용하도록 수정하거나 새로운 라이트 모드 색상 팔레트와 일치하도록 조정합니다.
3.  **디자인 일관성 검토**:
    -   모든 컴포넌트(버튼, 입력창, 모달 등)가 라이트 모드 디자인과 조화를 이루는지 확인하고 필요시 미세 조정합니다.
    -   가독성과 시각적 계층 구조가 명확하게 유지되는지 검토합니다.
