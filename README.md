# 간단 가계부 (Household Finance Manager)

이 프로젝트는 HTML, CSS, JavaScript만을 사용하여 만든 간단한 웹 기반 가계부 애플리케이션입니다. 사용자는 수입과 지출을 날짜별로 기록하고, 월별 및 연간 합계를 확인할 수 있습니다.

## 주요 기능

-   **수입 및 지출 입력**: 날짜를 지정하여 수입과 지출 내역을 추가합니다.
-   **월별 합계 보기**: 매월의 총수입, 총지출, 합계를 한눈에 볼 수 있습니다.
-   **연간 총합계**: 1년 전체의 재정 현황을 요약하여 보여줍니다.
-   **다단계 상세 내역**:
    1.  월별 합계 클릭 시, 날짜별로 요약된 수입/지출 내역을 표시합니다.
    2.  날짜별 요약에서 특정 금액을 클릭하면, 해당 날짜의 모든 개별 거래 내역을 확인할 수 있습니다.
-   **데이터 삭제**:
    -   개별 거래 내역을 하나씩 삭제할 수 있습니다.
    -   특정 날짜의 모든 수입 또는 지출 내역을 한 번에 삭제하는 '전체 삭제' 기능이 있습니다.
-   **데이터 영속성**: 모든 데이터는 브라우저의 `localStorage`에 저장되어, 창을 닫았다 열어도 내용이 유지됩니다.
-   **동적 UI 효과**: 버튼에 마우스를 올리거나 클릭할 때 부드러운 애니메이션 효과가 적용되어 사용자 경험을 향상시킵니다.

## 기술 스택

-   HTML
-   CSS
-   JavaScript (ES6+)
